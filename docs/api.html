API Reference

All endpoints use base URL: http://localhost:5000/api

Prediction & Chat

MethodEndpointDescription
POST/uploadUpload image (multipart/form-data), get disease prediction
POST/chatSend message, get chatbot response. Body: { message, sessionId, lang }
POST/translateTranslate text. Body: { text, targetLang } (hi, te)

Auth — Admin

MethodEndpointDescription
POST/admin/loginBody: { username, password }. Returns token.
GET/admin/verifyHeader: Authorization: Bearer <token>

Auth — Users (Farmers)

MethodEndpointDescription
POST/users/registerBody: { name, email, password, phone?, location? }
POST/users/loginBody: { email, password }
GET/users/profileHeader: Authorization: Bearer <token>
PUT/users/profileBody: { name?, phone?, location? }

Stats & Health

MethodEndpointDescription
GET/stats/publicPublic stats (totalPredictions, todayCount, totalUsers)
GET/dashboardAdmin stats (requires admin token)
GET/healthHealth check

AI Service (Python — port 8000)

MethodEndpointDescription
POST/predictImage file, returns disease prediction
GET/healthHealth and model status
