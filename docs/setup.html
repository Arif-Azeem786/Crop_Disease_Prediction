Setup Guide

Step-by-step instructions to run CropGuard locally.

Prerequisites

  • Node.js 18+
  • Python 3.10+
  • MongoDB (local or Atlas)
  • npm

1. AI Model Service

cd ai-model
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt
uvicorn main:app --reload --port 8000

Use numpy<2 if TensorFlow fails with NumPy 2.x.

2. Train Model (Optional, for real predictions)

Add images to ai-model/dataset/ per class. Then:

cd ai-model
python train_model.py

3. Backend

cd backend
npm install
# Create .env with MONGODB_URI, AI_SERVICE_URL, OPENAI_API_KEY
npm start

4. Frontend

cd frontend
npm install
npm start

5. Access

Open http://localhost:3000
