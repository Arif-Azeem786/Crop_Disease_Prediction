1. Overview & Tech Stack

CropGuard is a full-stack web application for detecting Banana and Wheat leaf diseases and assisting farmers with treatment advice in multiple languages.

LayerTechnology
FrontendReact, Tailwind CSS, Chart.js, React Router
BackendNode.js, Express
DatabaseMongoDB
AI ServicePython, FastAPI, TensorFlow (CNN)
ChatbotOpenAI API
VoiceBrowser SpeechSynthesis API

2. Setup Instructions

2.1 AI Model Service (Python)

cd ai-model
python -m venv venv
venv\Scripts\activate   # Windows
# source venv/bin/activate  # Linux/Mac
pip install -r requirements.txt
uvicorn main:app --reload --port 8000

Note: Without a trained model, the service runs in DEMO mode. For real predictions, train the model with python train_model.py after adding images to dataset/.

2.2 Backend (Node.js)

cd backend
npm install
# Create .env with MONGODB_URI, AI_SERVICE_URL, OPENAI_API_KEY
npm start

2.3 Frontend (React)

cd frontend
npm install
# Create .env with REACT_APP_API_URL=http://localhost:5000/api
npm start

Frontend runs at http://localhost:3000. Backend at http://localhost:5000. AI service at http://localhost:8000.

2.4 MongoDB

Ensure MongoDB is running (local or Atlas). Set MONGODB_URI in backend .env.
